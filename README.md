## 项目简介
在数字化浪潮中，传统文化往往面临“老龄化”或“过时”的刻板印象。**玄机黄历**旨在打破这一局面。它不仅仅是一个查询日历的工具，更是一个连接古今、融合虚实的文化载体。
项目采用纯原生 HTML/CSS/JS 开发，无需复杂框架依赖，通过 CDN 即可快速部署。设计上借鉴了“鸿蒙初开”的玄幻概念，结合玻璃拟态与动态特效，既保留了中老年人熟悉的黄历内容，又抓住了年轻人喜爱的“国潮”审美。

## 核心功能
### 1. 沉浸式入口体验
- **盘古开天**：用户进入时，首先看到的是混沌的星空与旋转的八卦。
- **开天辟地**：点击中心按钮，触发“天地分离”动画，寓意从混沌进入有序世界，极具仪式感。
### 2. 四大核心模块（独创玉佩/卷轴UI）
功能不再是枯燥的列表，而是通过独特的UI图案呈现：
- **天时（八卦圆形按钮）**：
    - **万年历**：公历农历对照，每日宜忌速查。
    - **每日运势**：基于日期的卦象推演，详尽的宜忌列表。
    - **择吉日**：嫁娶、搬家、开业等事项的吉日筛选。
- **地利（方形罗盘按钮）**：
    - **风水罗盘**：动态旋转的罗盘，展示今日吉神方位（喜神、财神、阳贵）。
    - **出行指南**：结合当日冲煞，提供出行建议。
- **人和（玉佩形状按钮）**：
    - **八字算命**：输入生辰，自动推算四柱八字、五行能量分布及命理简批。
    - **姓名测算**：基于笔画数理的姓名吉凶评分（三才五格）。
    - **生肖运势**：十二生肖的年度、月度运势详解。
- **玄机（卷轴形状按钮）**：
    - **解梦占卜**：输入梦境关键词，查询周公解梦寓意。
    - **六爻占卜**：模拟传统铜钱起卦，预测事物发展。
    - **祈福许愿**：电子祈福功能，满足精神寄托。
### 3. 动态视觉效果
- **五行能量流**：背景根据当日五行属性产生微妙的颜色流动。
- **粒子星空**：Canvas 绘制的高性能星空背景，闪烁效果随日期变化。
- **交互反馈**：所有按钮均配有悬浮发光、缩放或旋转动效。

## 技术架构
本项目坚持“轻量化、高性能、易部署”的原则。
- **前端骨架**：HTML5
- **样式引擎**：Tailwind CSS (CDN引入)
    - 使用了 Tailwind 的 JIT 模式理念，通过内联类名快速构建响应式布局。
    - 自定义配置扩展了 `cinnabar` (朱砂红)、`gold` (鎏金)、`jade` (碧玉) 等中国传统色。
- **逻辑交互**：原生 JavaScript (ES6+)
    - 无框架依赖，加载速度快，适合静态部署。
    - 模块化函数设计，便于后续维护。
- **图形渲染**：SVG & CSS3 Animation
    - 使用 SVG 绘制太极八卦图，体积小且可无限缩放。
    - 利用 CSS `@keyframes` 实现复杂的旋转、呼吸、光影动画。
- **字体资源**：Google Fonts
    - 引入 `Ma Shan Zheng` (马善政) 书法字体，营造古风韵味。

## 目标群体分析
### 1. 中老年群体（核心用户）
- **痛点**：传统纸质黄历查阅不便，现有APP广告多、字体小、操作复杂。
- **解决方案**：
    - **大字体与高对比度**：界面文字清晰，配色以金、红、黑为主，辨识度高。
    - **内容详尽**：包含传统的黄历宜忌、冲煞、吉神方位，符合日常参考习惯。
    - **操作直观**：简化导航，通过大色块按钮进入功能，降低学习成本。
### 2. 年轻群体（增长用户）
- **痛点**：对传统文化感兴趣，但排斥老旧的UI设计；追求“国潮”与“玄学”的新鲜感。
- **解决方案**：
    - **玄幻美学**：引入“鸿蒙”、“开天辟地”等玄幻概念，UI设计类似仙侠游戏。
    - **社交属性**：提供八字合婚、姓名测算、今日运势等易于分享的话题功能。
    - **交互体验**：流畅的动画、玻璃拟态设计，符合现代审美标准。

## 市场前景
1. **文化自信与国潮复兴**：
   随着国家对传统文化的扶持，年轻一代对“中式玄学”的接受度极高。星座塔罗之外，八字风水正在成为新的流量入口。
2. **老龄化社会的刚需**：
   中国老龄化程度加深，中老年人对黄历、运势类工具的依赖是长期且稳定的。一个界面友好、无广告干扰的纯净工具极具用户粘性。
3. **技术部署优势**：
   本项目为纯静态页面，可完美部署在 GitHub Pages、Vercel、Netlify 等免费平台，运维成本几乎为零。适合作为个人开发者练手项目或小型创业项目的 MVP（最小可行性产品）。

## 安装与使用
### 在线访问
直接将代码部署为静态网页即可访问。
### 本地运行
1.  克隆仓库：
    ```bash
    git clone https://github.com/your-username/yellow-calendar.git
    ```
2.  打开文件：
    直接在浏览器中打开 `index.html` 文件即可运行。
### 部署到 GitHub Pages
1.  在 GitHub 创建新仓库。
2.  上传 `index.html` 文件。
3.  在仓库设置中找到 Pages 选项，选择 Branch 为 main，保存即可获得在线访问链接。
